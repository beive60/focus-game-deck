# Tutorial Images

This directory contains screenshots and images for the Focus Game Deck tutorial.

## Required Images

The following images should be placed in this directory:

- `tutorial-welcome.png` - Welcome screen/overview
- `tutorial-game-registration.png` - Game registration interface
- `tutorial-app-registration.png` - Managed apps configuration
- `tutorial-obs-integration.png` - OBS integration setup
- `tutorial-game-launching.png` - Game launching demonstration

## Image Requirements

- Format: PNG
- Recommended size: 1200x800 pixels or similar aspect ratio
- File size: Keep under 500KB per image for fast loading

## Creating Tutorial Images

1. Take screenshots of the ConfigEditor in action
2. Annotate key features if needed
3. Save with the filenames listed above
4. Place in this directory

If images are not present, the tutorial will still work but will not display screenshots.
