Focus Game Deck

Unified Gaming Environment Automation Tool

Focus Game Deck is a free, open-source unified gaming environment automation tool. With a single executable, you can configure games through an intuitive GUI or launch them directly from the command line. It prevents anti-cheat conflicts, configures gaming-optimized settings (like disabling Windows key), and integrates with OBS, Discord, and VTube Studio to create the perfect gaming environment automatically. Built with PowerShell and WPF, it uses JSON configuration files for settings and provides a seamless user experience.

Focus Game Deck is widely used in industries such as competitive gaming, game streaming, and software development for gamers who require optimization tools. As well as being employed commercially, it is also available as a free-to-use tool under the MIT license for personal and educational use.

Download

Focus Game Deck is available for Windows 10/11 as a simple, lightweight executable file (just a few MB). Simply double-click to open the intuitive configuration interface, and of course, you can also launch games directly from the command line if preferred. The latest release and source code are available through GitHub.

Features and Capabilities

One of the key features of Focus Game Deck is its support for extensive gaming platform integration. The software supports multiple game launchers and automated optimization processes that can be run simultaneously to provide a comprehensive gaming environment management solution. In addition, it offers a wide range of optimization features for various gaming scenarios and tasks that need to be performed on a regular basis.

  • Unified Interface - single executable with integrated GUI configuration and direct game launching
  • Process Management - temporarily disable conflicting processes during gaming sessions
  • Anti-cheat Compatibility - prevent common conflicts with anti-cheat systems like BattlEye and EasyAntiCheat
  • Platform Integration - support for Steam, Epic Games Store, and Riot Client
  • OBS Integration - automatic OBS scene switching and recording management
  • Discord Integration - status updates and Rich Presence integration
  • VTube Studio Support - model switching and scene management for streamers
  • System Optimization - Windows key disabling, power plan switching, and resource optimization
  • Configuration Management - JSON-based configuration with validation and backup
  • Logging - comprehensive logging and error reporting for troubleshooting
  • Automatic Restoration - restore all settings and processes after game sessions end
