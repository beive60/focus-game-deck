自動最適化

アンチチートシステムと競合する可能性のあるプロセスやWinキーを自動で無効化。ゲーム起動エラーを防ぎ、システムリソースも最適化。ゲーム終了後は自動的に元の環境に復元されます。

軽量・高速動作

PowerShellベースの軽量設計により、システムリソースをほとんど消費せず、瞬時に環境最適化を実行します。

各種ゲームライブラリ連携

現在Steam、Epic Games Store、Riot Clientに対応。各プラットフォームのゲームを検出し、ゲーム起動と連動して環境最適化を実行します。将来的により多くのプラットフォームに対応予定です。

VTube Studio連携

ゲーム開始時にVTuberアバターを自動でゲーミングモードに切り替え、配信者の環境を最適化します。Steam版・スタンドアロン版両対応。

Discord環境最適化

ゲーム開始時に自動でゲーミングモードに切り替え、オーバーレイ無効化やステータス変更で集中できる環境を構築します。

OBS自動連携

OBSのリプレイバッファを自動で開始・停止。ゲーム中の神プレイを逃すことなく、手動操作の必要もありません。

なぜFocus Game Deckなのか？

01

オールインワン統合

OBS、Discord、VTube Studio、NoWinKeyなど6つ以上のアプリケーションを統合管理。複数のツールを個別に操作する手間を完全に省き、ワンクリックで理想のゲーミング環境を構築します。

02

軽量で高速

わずか2.5MBの軽量サイズでありながら、PowerShellの高速性を活かして瞬時に環境を最適化。システムリソースをほとんど消費せず、ゲームのパフォーマンスを妨げません。

03

幅広いプラットフォーム対応

Steam、Epic Games Store、Riot Clientの3つの主要プラットフォームに対応。どのゲームランチャーを使っていても、一貫した最適化体験を提供します。

よくある質問

Focus Game Deckに関する疑問にお答えします

Focus Game Deckは本当に無料ですか？

+

はい、完全に無料です。MITライセンスのもとでオープンソースとして公開されており、すべての機能を制限なく利用できます。今後も有料化の予定はありません。

どのようなゲームに対応していますか？

+

現在Steam、Epic Games Store、Riot Clientに対応しています。これらのプラットフォームのゲームを自動検出し、連携できます。今後のアップデートでOrigin、Battle.net、GOGなど、他のプラットフォームとの連携も強化予定です。

システムに悪影響はありませんか？

+

Focus Game Deckは軽量なPowerShellスクリプトベースで動作し、システムリソースをほとんど消費しません。また、すべての変更は一時的で、ゲーム終了後に元の状態に自動復元されます。

アンチチートシステムとの競合問題は解決されますか？

+

はい。多くの競技ゲーマーが直面する「日常使いのソフトウェアがアンチチートシステムに検知されてゲームが起動できない」問題を解決します。Focus Game Deckは、競合の可能性があるプロセスを一時的に終了し、ゲーム終了後に自動復元します。

設定は難しくないですか？

+

基本的な使用では特別な設定は不要です。より詳細なカスタマイズはconfig.jsonファイルで行えますが、近日中にGUIエディターをリリース予定で、さらに簡単になります。

ウイルス対策ソフトで誤検知されることはありますか？

+

PowerShellスクリプトは稀に誤検知される場合があります。ソースコードは完全に公開されており、透明性を確保しています。信頼できるウイルス対策ソフトでは問題ありません。

サポートやコミュニティはありますか？

+

GitHubのIssuesページでバグ報告や機能リクエストを受け付けています。また、コミュニティによる貢献（翻訳、ドキュメント改善など）も大歓迎です。
