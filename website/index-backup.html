What It Does

Technical features and integrations

Automatic Optimization

Automatically disables processes and Windows keys that may conflict with anti-cheat systems. Prevents game launch errors and optimizes system resources. The environment is automatically restored after game completion.

Lightweight & High-Speed

The lightweight PowerShell-based design consumes almost no system resources and executes environment optimization instantly.

Game Library Integration

Currently supports Steam, Epic Games Store, and Riot Client. Detects games from each platform and executes environment optimization in conjunction with game launch. Future support for more platforms is planned.

VTube Studio Integration

Automatically switches VTuber avatars to gaming mode when games start, optimizing the streamer's environment. Supports both Steam and standalone versions.

Discord Environment Optimization

Automatically switches to gaming mode when games start, disabling overlays and changing status to create a focused environment.

OBS Auto Integration

Automatically starts and stops OBS replay buffer. Never miss epic gameplay moments without manual operation.

Frequently Asked Questions

Answers to your questions about Focus Game Deck

Is Focus Game Deck really free?

+

Yes, it's completely free. It's published as open source under the MIT license, and you can use all features without any restrictions. There are no plans to charge for it in the future.

What games does it support?

+

Currently supports Steam, Epic Games Store, and Riot Client. It automatically detects and integrates with games from these platforms. Future updates are planned to enhance integration with other platforms like Origin, Battle.net, and GOG.

Does it have any negative impact on the system?

+

Focus Game Deck operates based on lightweight PowerShell scripts and consumes almost no system resources. Additionally, all changes are temporary and automatically restored to the original state after game termination.

Are anti-cheat system conflicts resolved?

+

Yes. It solves the problem many competitive gamers face where "everyday software is detected by anti-cheat systems and prevents games from launching." Focus Game Deck temporarily terminates processes that may conflict and automatically restores them after the game ends.

Is the setup difficult?

+

No special configuration is required for basic use. More detailed customization can be done via the config.json file, but a GUI editor will be released soon to make it even easier.

Are there false positives with antivirus software?

+

PowerShell scripts may rarely trigger false positives. The source code is completely open, ensuring transparency. There are no issues with trusted antivirus software.

サポートやコミュニティはありますか？

+

GitHubのIssuesページでバグ報告や機能リクエストを受け付けています。また、コミュニティによる貢献（翻訳、ドキュメント改善など）も大歓迎です。
