Quick Start Guide

Focus Game Deck v2.0.0 introduces a unified interface that combines configuration management and game launching in a single, user-friendly application.

Launching the Application

GUI Mode (Default)

Simply double-click Focus-Game-Deck.exe to open the configuration interface.

Screenshot: Main application window

Command Line Mode

Run Focus-Game-Deck.exe <game-id> to launch games directly from command line.

Screenshot: Command line usage

Game Launcher Tab

The Game Launcher tab is your primary interface for launching configured games with optimized settings.

Screenshot: Game Launcher tab interface

Using the Game Launcher

  1. Select your game from the available game cards
  2. Review the game status and launch button
  3. Click "Launch Game" to start with optimized environment
  4. The application will automatically configure gaming settings
  5. Game will launch with anti-cheat protection and optimizations

Game Launcher Features

  • One-click game launching with environment optimization
  • Real-time status indicators for each game
  • Integration with Steam, Epic Games Store, and Riot Client
  • Automatic settings restoration after gaming sessions

Game Settings Configuration

Configure individual games with specific launch parameters, process management, and platform integration.

Screenshot: Game Settings tab

Adding a New Game

  1. Navigate to the "Game Settings" tab
  2. Click "Add New Game" button
  3. Enter game details (Name, Steam App ID, Process Name)
  4. Configure platform-specific settings if needed
  5. Save configuration to make the game available in launcher

Configuration Options

Basic Settings

  • Game Name: Display name in launcher
  • Steam App ID: For Steam integration
  • Process Name: Executable filename

Advanced Settings

  • Launch Arguments: Custom startup parameters
  • Platform: Steam, Epic, Riot, or Standalone
  • Pre-launch Actions: Environment preparation

Troubleshooting

Common Issues and Solutions

Game Not Launching

  1. Verify game configuration in Game Settings tab
  2. Check Steam App ID or executable path
  3. Ensure platform (Steam/Epic/Riot) is running
  4. Check application logs for error messages

OBS Connection Issues

  1. Verify OBS Studio is running with WebSocket plugin
  2. Check Host and Port settings in Global Settings
  3. Ensure WebSocket password matches configuration
  4. Test connection from OBS WebSocket settings

Anti-cheat Compatibility

Focus Game Deck is designed to be anti-cheat friendly. If you encounter issues:

  • Ensure you're using the digitally signed release version
  • Add Focus Game Deck to your antivirus whitelist
  • Report any false positives to the development team
