パワフルな機能

すべてのアクションがワンクリックで完了

自動最適化

アンチチートシステムと競合する可能性のあるプロセスやWinキーを自動で無効化。ゲーム起動エラーを防ぎ、システムリソースも最適化します。

超高速起動

PowerShellの軽量性を活かし、システムに負荷をかけることなく瞬時に環境を最適化します。

Steam連携

Steamライブラリのゲームを検出し、ゲーム起動と連動して環境最適化を実行します。

ゲーム終了後の自動復元

ゲーム終了後は自動的に元の環境設定に復元され、日常使いに影響を与えません。

アンチチート競合回避

日常使いソフトがアンチチートシステムに検知される問題を解決。競合プロセスを一時終了し、ゲーム終了後に自動復元します。

簡単設定

JSON設定ファイルで細かくカスタマイズ可能。近日中にGUIエディターもリリース予定。

なぜFocus Game Deckなのか？

01

競技レベルのパフォーマンス

不要なプロセスを自動終了し、システムリソースを最大限ゲームに集中させます。また、日常使いのソフトウェアがアンチチートシステムと競合してゲームが起動できない問題も解決します。

02

ストレスフリーな体験

「ゲームが起動しない」というストレスからも解放。Winキー無効化や競合プロセス終了など、ゲーム前の面倒な準備やゲーム後の後片付けは完全に自動化。純粋にゲームに集中できます。

03

時間の節約

毎回5-10分かかっていた環境設定が、ワンクリックで完了。その時間をスキル向上に使えます。

よくある質問

Focus Game Deckに関する疑問にお答えします

Focus Game Deckは本当に無料ですか？

+

はい、完全に無料です。MITライセンスのもとでオープンソースとして公開されており、すべての機能を制限なく利用できます。今後も有料化の予定はありません。

どのようなゲームに対応していますか？

+

現在はSteamライブラリのゲームと連携できます。設定によりSteam以外のゲームでも利用可能です。今後のアップデートでEpic Games Store、EA App、Riot Clientなど、他のプラットフォームとの連携も強化予定です。

システムに悪影響はありませんか？

+

Focus Game Deckは軽量なPowerShellスクリプトベースで動作し、システムリソースをほとんど消費しません。また、すべての変更は一時的で、ゲーム終了後に元の状態に自動復元されます。

アンチチートシステムとの競合問題は解決されますか？

+

はい。多くの競技ゲーマーが直面する「日常使いのソフトウェアがアンチチートシステムに検知されてゲームが起動できない」問題を解決します。Focus Game Deckは、競合の可能性があるプロセスを一時的に終了し、ゲーム終了後に自動復元します。

設定は難しくないですか？

+

基本的な使用では特別な設定は不要です。より詳細なカスタマイズはconfig.jsonファイルで行えますが、近日中にGUIエディターをリリース予定で、さらに簡単になります。

ウイルス対策ソフトで誤検知されることはありますか？

+

PowerShellスクリプトは稀に誤検知される場合があります。ソースコードは完全に公開されており、透明性を確保しています。信頼できるウイルス対策ソフトでは問題ありません。

サポートやコミュニティはありますか？

+

GitHubのIssuesページでバグ報告や機能リクエストを受け付けています。また、コミュニティによる貢献（翻訳、ドキュメント改善など）も大歓迎です。
