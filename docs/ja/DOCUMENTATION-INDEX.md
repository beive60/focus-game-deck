# Focus Game Deck - ドキュメント索引

## 📚 プロジェクト概要

Focus Game Deckは、競技PCゲーマーのための**ゲーミング環境自動化ツール**です。ゲーム起動時に関連アプリケーションを自動制御し、集中できる環境を瞬時に構築します。

### 🎯 基本理念

- **オープンで歓迎的なコミュニティ**の構築
- **MITライセンス**による最大限の自由度
- **セキュリティファースト**：アンチチート誤検知リスクの最小化
- **透明性の確保**：完全オープンソース
